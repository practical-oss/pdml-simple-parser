# Simple Core PDML Parser

A simple, minimalist, dependency-free, recursive descent LL(1) parser for [Core PDML](https://pdml-lang.dev/docs/core/specification/index.html), written in Java.

[PDML extensions](https://pdml-lang.dev/docs/extensions/user_manual/index.html) are not supported.

File `Start.java` in the source code tree contains a basic usage example.

Note: A significantly more advanced implementation, supporting all PDML extensions, is available in the [pdml-ref-impl](https://github.com/practical-oss/pdml-ref-impl) repository.
